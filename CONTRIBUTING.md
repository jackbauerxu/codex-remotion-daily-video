# Contributing

Keep this repository useful as a production Skill, not only a workflow note.

## Rules

- Keep `codex-remotion-daily-video/SKILL.md` short and navigational.
- Put detailed production rules in `codex-remotion-daily-video/references/`.
- Put runnable starter code in `examples/remotion-starter/`.
- Add or update `test-prompts.json` when changing trigger behavior.
- Run `npm run verify` before submitting changes.
- Do not claim rendered video output unless the artifact exists and passes verification.

## Change Scope

Use `codex-workflow-builder` for parent workflow design. Use this repository for video-specific production logic, Remotion templates, JSON schemas, HyperFrames promotion rules, and QA.
